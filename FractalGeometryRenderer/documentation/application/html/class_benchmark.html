Detailed Description

A class used for benchmarking pieces of code.

After creating an instance of this class, the start method must be called before calls to addMarkerNow and recordFrameTime will succeed. The stop method should be called when finished.

This class will automatically call the method printResultsToConsole when its destructor is called.
