Detailed Description

Class used for storing keyboard and mouse events that have occured.

Events include:

  • Keyboard
    1. Move camera forward
    2. Move camera backward
    3. Move camera left
    4. Move camera right
    5. Move camera up
    6. Move camera down
  • Mouse
